USAGE EXAMPLES
=============================================================================

Start System:
  $ ./start-monitoring.sh

Access Dashboards:
  $ open http://localhost:3001              # Web UI
  $ ./monitoring/cli/monitor-cli.sh watch   # CLI

Query API:
  $ curl http://localhost:3000/api/status | jq

Send Notifications:
  $ ./monitoring/notifications/notify.sh test slack

Collect Metrics:
  $ python3 monitoring/backend/metrics-collector.py report --session SESSION_ID

Stop System:
  $ ./stop-monitoring.sh
